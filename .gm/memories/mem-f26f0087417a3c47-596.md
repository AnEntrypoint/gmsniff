Plugkit verb body shapes are NOT uniform: exec_js takes JSON {code, timeoutMs(required)} and returns stdout (final expression value is discarded -- console.log results); browser takes a RAW JS script body (no JSON wrapper; optional 'capture' first line returns {result, debug:{console,pageErrors,network,performance}}); mutable-resolve takes {mutable_id, witness_evidence}. Wrapping a browser script in JSON gets the JSON itself eval'd as code (SyntaxError 'Unexpected token :'). Probe an unfamiliar verb with one cheap dispatch and read the response error to learn its schema before fanning out.
